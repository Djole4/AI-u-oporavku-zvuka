## Napomena:

- Video fajlovi moraju biti u MP4 formatu
- Preporučena rezolucija: 1920x1080 (Full HD)
- Preporučeni codec: H.264
- Video će se automatski prikazati na sajtu

## Kako dodati video:

1. Pretvori svoj video u MP4 format (ako već nije)
2. Preimenuj ga tačno kako je navedeno gore
3. Kopiraj u ovaj folder
4. Refresh-uj stranicu
